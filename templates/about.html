Data Sources

1. College Scorecard (Dataset published by the USA Department of Education).
This dataset contains over 1,800 features about academic, admissions, completion, student body, and many other categories for over 7,000 college institutions in the United States. In addition, the dataset is organized for each academic year, starting in 1996-1997 and ending in 2015-2016.

2. Twitter (Twitter API).
I intend to collect a dataset containing tweets with geographical and date information such that I can estimate the amount of social network usage per capita segmented by location (state-wide) and time (year-wide).

Project Objectives

☑ Download, clean up, and explore the College Scorecard dataset.

☑ Perform a detailed data analysis in the College Scorecards dataset.

☑ Develop a webpage to organize and present the main aspects of the project.

☑ Deploy the project webpage to the internet using Heroku.

☐ Implement a machine learning alogirthm (linear regression) to predict college completion rates using the College Scorecard main features.

☐ Use Twitter API to gather an appropiate dataset containing tweets with geographical and date tags.

☐ Perform an statistical test to determine the correlation between Twitter usage and college completion rates.

☐ Future: Examine correlations with other datasets (Videogames, Wealth, Crime, Weather, etc.).
